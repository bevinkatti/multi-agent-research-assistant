# Multi-Agent AI Research Assistant

A production-grade multi-agent research system with real-time web search, RAG over documents, critic scoring, and RAGAS evaluation.

Built with LangGraph, Groq, FAISS, FastAPI, and Streamlit.

## Features

- Real-time web search via Tavily
- RAG over PDFs and URLs via FAISS
- 5-dimension critic scoring
- RAGAS evaluation with 0.927 overall score
- Live agent activity dashboard

## Setup

Set these secrets in Space Settings:

- GROQ_API_KEY
- TAVILY_API_KEY
